# Punisher - Privacy-First Bitcoin AI Assistant

A locally-hosted, open-source personal AI assistant focused on **privacy**, **Bitcoin**, and **market intelligence**. Runs 100% on bare metal Ubuntu, no cloud dependencies.

## Features
- **100% Local**: No cloud dependencies.
- **Interfaces**: Telegram, CLI, and Brutalist Web Dashboard.
- **Bitcoin Focus**: Whale watching, market sentiment, trader cohort analysis.
- **Continuous Research**: Auto-updates from local search engine.
- **Browser Automation**: Undetectable automation using `nodriver`.

## Setup
1. Install dependencies: `uv sync`
2. Run server: `uv run punisher-server`
3. Run CLI: `uv run punisher`
